Algorithms Used:

Naive Bayes

Navie Bayes methods are a set of supervised learning algorithms based on applying Bayes' theorem with the “naive” assumption of conditional independence between every pair of features given the value of the class variable.



Equations Used:

Pr(W|S), Pr(W|H), Pr(S|W), Accuracy, Precision

Pr(W|S) = of spam files containing W/of spam files

Pr(W|H) = of ham files containing W/of ham files

Pr(S|W) = Pr(W|S)/Pr(W|S) + Pr(W|H)

Accuracy = numTruePositives + numTrueNegatives/numFiles

Precision = numTruePositives/numFalsePositives + numTruePositives



Efficiency of Program:

O(N * L * log(N * L)) and O(N * L + M)

The time and space complexity of the SpamDetector class is O(N * L * log(N * L)) and O(N * L + M), respectively, where N is the total number of files in the training and test set, L is the length of the longest file, and M is the number of distinct words in the training set.
